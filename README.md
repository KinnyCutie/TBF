# TBF
Twitch Bot Flagger, It flags bots on twitch using a very simple age-based check.

How to use:

1. Download and enable the extension/Add it to your browser
2. On twitch.tv, you will now see a [!BOT] [?!BOT] or [?BOT] depending on the time the account was created, it only checks age
You shouldn't 100% base a ban off this age check, but it can be useful if a user seems suspicious
